# Back Bot
### A bot for the best joke of all time

## Dependencies
* Python 3.4.2+
* discord.py
* PyNaCl
* ffmpeg

#### Install discord.py

```shell_session
  $ pip install discord
```

#### Install pynacl

```shell_session
  $ pip install pynacl
```

#### Install ffmpeg

    Good luck!
    Make sure the path environment variable points to the executable folder!

## Run

```shell_session
  $ python back_bot.py
```


The file will be located from the base directory.
